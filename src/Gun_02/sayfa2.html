1-Selenium bir programlama dili içinde tarayıcıyı kontrol etmek için kütüphane hizmeti sunan automation tool dur.



2-Automation test, manuel test sonrası ve artık yazılıma ekleme yapılmayacağı zaman %90 oranında automate edilebilir.



3-Selenium, chrome,firefox,opera ve yandex browserlarda çalışır.



4-Selenium is an open-source tool that automates web browsers. It provides a single interface that lets you write test scripts in programming languages like Ruby, Java, NodeJS, PHP, Perl, Python, and C#, among others.



5-SDLC (SOFTWARE/SYSTEMS DEVELOPMENT LIFE CYCLE) NEDİR? SOFTWARE DEVELOPMENT LIFE CYCLE; Yazılım geliştirme sürecinde birçok kurumun haberi olmadığı veya önemsemediği için atladığı fakat yapılacak yazılımın kalitesini belirleyen en önemli süreçlerden birisidir.
